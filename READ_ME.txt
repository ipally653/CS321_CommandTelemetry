The XBEE_READER.py program requires the following packages
which can all be installed via pip:

pyserial
keyboard
digi-xbee

To use the program, simply run it as you would a normal python program.

To send the cut command, hit the 'Ctrl' and 'Space' keys simultaneously,
and you should see a message pop up telling you that the cut command has
been sent.

To end the program, hit the '`' key
